This tool automatically converts any "#" strings to use the Unit Vehicle "UVString()" format. It allows workshop-uploaded content to not have common localization errors, such as added spaces at the end of strings.

---> Requirements <---
Python 3.11 or newer

---> How do I use this? <---
1) Move the .py file into the folder with your .lua files you wish to convert.
2) Run the .py file!
3) Type the number for the file you wish to convert and hit enter.
4) Repeat step 3 for every file you wish to convert.
